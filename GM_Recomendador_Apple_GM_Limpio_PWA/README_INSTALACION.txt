Cómo usar en teléfono:
1. Subí esta carpeta completa a Netlify Drop, Vercel o GitHub Pages.
2. Abrí el link desde el celular.
3. En iPhone: Safari > Compartir > Agregar a pantalla de inicio.
4. En Android: Chrome > Instalar app / Agregar a pantalla de inicio.
