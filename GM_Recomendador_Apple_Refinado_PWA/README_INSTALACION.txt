GM Recomendador Apple PWA

Cómo usar:
1. Subí esta carpeta completa a Netlify Drop, Vercel o GitHub Pages.
2. Abrí el link desde iPhone o Android.
3. En iPhone: Safari > Compartir > Agregar a pantalla de inicio.
4. En Android: menú del navegador > Instalar app / Agregar a pantalla de inicio.

Archivos:
- index.html: estructura principal.
- styles.css: estética mobile-first tipo Apple + branding GM.
- app.js: base de unidades, scoring y lógica de recomendación.
- manifest.webmanifest y service-worker.js: soporte PWA.
